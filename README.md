# 🗳️ Online Voting System

A web-based Online Voting System developed using Java, JSP, Servlets, and MySQL. The system allows voters to securely cast their votes while ensuring that each voter can vote only once.

## 📌 Project Overview

The Online Voting System digitizes the election process by providing a secure and user-friendly platform for voter registration, authentication, vote casting, and result management.

This project was developed as a college project to demonstrate web application development using Java EE technologies.

---

## ✨ Features

### Voter Module

* Voter Registration
* Secure Login Authentication
* Profile Picture Support
* Cast Vote
* One Vote Per Voter Restriction
* View Election Status

### Admin Module

* Admin Login
* Add Candidates
* Edit Candidate Details
* Delete Candidates
* Manage Elections
* View Voting Results

### System Features

* Secure Database Storage
* Responsive User Interface
* Session Management
* Vote Timestamp Recording
* Duplicate Voting Prevention

## ⚙️ Database

Main Tables:

* voters
* candidates
* votes
* admins

### Voting Logic

* Each voter has a unique voter number.
* A voter can cast only one vote.
* Votes are recorded with timestamps.
* Results are generated from stored votes.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/vids1538/online-voting-system
```

2. Import the project into Eclipse.

3. Configure Apache Tomcat.

4. Create the MySQL database.

5. Update database credentials in the connection configuration file.

6. Run the project on Tomcat.

## Day 2 (Date : 20/04/2026)

### MySQL Installation & Error Fix

### 🔹 Task

* Attempted to install MySQL Server on my system.
* Added .jar file into IntelliJ using Project Structure → Libraries


### ⚠️ Problem Faced

While installing MySQL, I encountered **Error 2503**, which is related to **Windows Installer permission issues**.
The installer was unable to proceed due to insufficient administrative access.


### 🔍 What I Did

* Researched the error online
* Watched YouTube tutorials to understand the issue
* Learned that the problem occurs due to lack of proper permissions


### ✅ Solution Applied

* Opened **Command Prompt as Administrator**
* Used `msiexec` command to run the installer
* Successfully bypassed the permission issue


### 🎯 Result

* MySQL installation started successfully after applying the fix.
* Successfully added JDBC driver ,Project is now ready to connect Java with MySQL

### 📚 Learning


* Running installers via CMD is a powerful solution
*  services like net start mysql80 ,net stop mysql80
* login to mysql and monitor mysql 
* .jar file acts as a bridge between Java and MySQL
